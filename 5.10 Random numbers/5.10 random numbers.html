5.10 Random Numbers

Notes

Intro

Random Numbers are useful in software development for many reasons - game mechanics, generating usernames/passwords, anytime you need something seemingly unpredictable
Pseudo-random numbers - computers are just complex calculators that can't create truly random numbers, but instead, use an algorithm that generates numbers that seem random

Random numbers in JS

Javascript provides a number of functions that are useful for generating random numbers

  • Math.random() is a function that returns a pseudo-random number between 0 and 1, not including 1
  • Math.floor() is a function that takes a decimal number and rounds it down - chops off the decimal part

Random number calculations

To produce a random integer from min to max (including min and excluding max) use this calculation.
Math.floor(Math.random() * (max - min)) + min

Example Generate a random number from 1 to 10 and store it in a variable.
var randNum = Math.floor(Math.random) * 10) + 1;
